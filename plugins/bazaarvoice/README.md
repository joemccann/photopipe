Bazaarvoice Config
=

You will need to do two things:

1. Create a ```bv-config.json``` file in this directory.
2. Inside the file place the following, but put in your credentials

```
{
    "apiversion": "5.2"
  , "passkey": "YOUR_ACTUAL_PASSKEY"
  , "userId": "YOUR_ACTUAL_USERID"
  , "url": "YOUR_ACTUAL_URL"
}
```
